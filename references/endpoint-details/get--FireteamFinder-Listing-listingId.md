# Fireteamfinder Getlisting

## Overview
Retrieves a Fireteam listing.

## Endpoint Details
- **HTTP Method:** GET
- **Path:** `/FireteamFinder/Listing/{listingId}/`
- **Operation ID:** `FireteamFinder.GetListing`
- **Tags:** FireteamFinder
- **Deprecated:** false

## Authentication
- **Required:** No\n- **Permissions:** None

## Parameters

### Path Parameters
| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| listingId | integer (int64) | Yes | The ID of the listing to retrieve. |
