# Destiny2 Getpublicmilestonecontent

## Overview
Gets custom localized content for the milestone of the given hash, if it exists.

## Endpoint Details
- **HTTP Method:** GET
- **Path:** `/Destiny2/Milestones/{milestoneHash}/Content/`
- **Operation ID:** `Destiny2.GetPublicMilestoneContent`
- **Tags:** Destiny2
- **Deprecated:** false

## Authentication
- **Required:** No\n- **Permissions:** None

## Parameters

### Path Parameters
| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| milestoneHash | integer (uint32) | Yes | The identifier for the milestone to be returned. |
